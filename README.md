# SUMAGO_TASKS

## Object Oriented Programming (OOP) Assignment

This repository contains Python programs demonstrating:

- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Constructor Reusability using super()

1. Food Delivery Application
2. Bank Management System
3. Employee Payroll System
4. Hospital Billing System
5. Online Payment Gateway
6. OTT Subscription System
7. Vehicle Rental System
8. Student Result Management System
9. E-Commerce Order System
10. Movie Ticket Booking System
